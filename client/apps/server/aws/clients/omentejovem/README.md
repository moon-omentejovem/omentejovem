# omentejovem Infrastructure

This folder represents the necessary infrastructure to deploy the backend of the omentejovem applications in the AWS.

The backend is running on an EC2 instance with Apache and Wordpress.

AWS Account ID: 732075124266

# Access the server

To access the server you must have the public and private key. If you don't have the private key in your machine, contact your administrators.

ssh -i ~/.ssh/omentejovem-ec2-key.pem SYSUSER@IP_ADDRESS_OF_SERVER
